# FirPE 帮助文档

## 介绍

使用 VitePress 搭建的 FirPE 帮助文档

## 软件架构

使用 VitePress 生成

## 安装教程

1. 安装[Node.js](https://nodejs.org/zh-cn/)
2. 安装依赖: `npm install`

## 使用说明

- 本地预览: `npm run docs:dev`
- 编译打包: `npm run docs:build`
